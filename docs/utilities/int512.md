## Overview

`Int512` is a dependency-free signed 512-bit integer for exact predicates and
other calculations that need several times the width of `long long` but do not
need a general arbitrary-precision integer. It stores eight little-endian
64-bit limbs and uses two's-complement arithmetic.

The alias `i512` names the same type. Arithmetic is performed modulo $2^{512}$,
and comparisons and decimal output interpret the result as a signed value in
$[-2^{511}, 2^{511}-1]$.

Use [`BigInt`](bigint.md) instead when division or unbounded precision is
required.

## Interface

| Method / operator | Description | Time | Memory |
| --- | --- | --- | --- |
| `Int512()` | Constructs zero. | $O(1)$ | $O(1)$ |
| `Int512(Integer value)` | Sign-extends or zero-extends an integral type of at most 64 bits. | $O(1)$ | $O(1)$ |
| `is_zero()` | Tests whether the value is zero. | $O(1)$ | $O(1)$ |
| `is_negative()` | Tests the sign bit. | $O(1)$ | $O(1)$ |
| `sign()` | Returns `-1`, `0`, or `1`. | $O(1)$ | $O(1)$ |
| unary `+`, unary `-` | Returns the value or its two's-complement negation. | $O(1)$ | $O(1)$ |
| `+`, `-`, `+=`, `-=` | Adds or subtracts modulo $2^{512}$. | $O(1)$ | $O(1)$ |
| `*`, `*=` | Multiplies modulo $2^{512}$. | $O(1)$ | $O(1)$ |
| `<`, `>`, `<=`, `>=`, `==`, `!=` | Compares signed values. | $O(1)$ | $O(1)$ |
| `to_string()` | Returns the signed decimal representation. | $O(1)$ | $O(1)$ |
| `operator<<` | Writes the signed decimal representation. | $O(1)$ | $O(1)$ |

The stated bounds are constant because the width is fixed. Multiplication uses
64 limb products, and decimal conversion emits at most 155 digits.

`Int512` intentionally does not provide division, remainder, parsing, or bit
operations. This keeps the header small for its main use as an exact
intermediate type.

## Example

```cpp
#include "utilities/int512.hpp"

#include <iostream>

int main() {
    using m1une::utilities::Int512;

    Int512 value = 4'000'000'000'000'000'000LL;
    Int512 square = value * value;
    std::cout << square << "\n";
}
```
