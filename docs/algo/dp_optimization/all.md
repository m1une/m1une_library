## Overview

`algo/dp_optimization/all.hpp` includes helpers for optimizing dynamic
programs and structured implicit matrices.

Public namespaces stay short: Alien Trick helpers use `m1une::dp`, and Monge
helpers use `m1une::monge`.

## Included Headers

| Header | Contents |
| --- | --- |
| `algo/dp_optimization/alien_trick.hpp` | Exact-count optimization through Lagrangian relaxation. |
| `algo/dp_optimization/monge/all.hpp` | SMAWK, monotone minima, Knuth/D&C DP optimization, LARSCH, Monge checks, and structured min-plus/max-plus convolution. |
